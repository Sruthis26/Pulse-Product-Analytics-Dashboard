# Pulse — Product Analytics Dashboard

A modern AI-assisted Product Analytics Dashboard built to simulate how real product and growth teams analyze user behavior, retention, funnels, churn, and experimentation metrics.

Designed as a product analytics operating system for a fictional music streaming platform called **StreamFlow**.

## Overview

Pulse simulates real-world product analytics workflows used by:
- Product Managers
- Growth Teams
- Product Analysts
- Retention Teams
- SaaS Companies

The platform helps analyze:
- KPI performance
- user activation funnels
- churn behavior
- cohort retention
- experimentation outcomes
- product growth opportunities

The project uses dynamic simulated data to create realistic analytics snapshots without requiring backend infrastructure.

# Features

## KPI Dashboard
- Daily Active Users (DAU)
- Monthly subscribers
- Trial-to-paid conversion
- Session duration
- ARPU tracking
- AI-generated product insights

---

## Funnel Analytics
Analyze user journeys such as:
- App Install → Activation
- Trial → Paid
- Referral Journey
- Playlist Creation

Includes:
- conversion tracking
- drop-off analysis
- friction detection
- AI recommendations

---

## Churn Analytics
Track:
- churn trends
- at-risk users
- retention opportunities
- churn by user plans
- churn hypotheses

Includes AI-generated retention insights and behavioral analysis.

---

## Cohort Analysis
Visualize:
- retention curves
- cohort stickiness
- long-term user retention
- cohort heatmaps

Designed to simulate real SaaS/product retention workflows.

---

## A/B Testing Dashboard
Simulated experimentation platform featuring:
- variant comparison
- confidence scores
- conversion uplift
- estimated MRR impact
- experiment intelligence

Examples:
- paywall timing tests
- onboarding experiments
- homepage layout experiments

---

## AI Product Recommendations
Pulse generates simulated AI-style insights such as:
- activation bottlenecks
- retention opportunities
- growth recommendations
- experiment prioritization
- product optimization suggestions
